# Postman API Testing Assignment

## Overview
This project demonstrates API testing using Postman.

## Features
- GET, POST, PUT, PATCH, DELETE methods
- Environment variables
- Dynamic request data
- Assertions using Chai
- API chaining
- Collection Runner execution

## Tools Used
- Postman
- GitHub

## How to Use
1. Import the Postman collection JSON file
2. Set environment variables
3. Run collection using Postman Runner
